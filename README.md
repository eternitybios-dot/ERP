# 反応しない練習帳

強迫性障害（OCD）とチック症のための、スマホ用セルフヘルプPWAです。

不安や不快感を「下げる」ためのアプリではありません。強迫行為・確認・打ち消し・クセに**「反応しなかった」「別の動きに置き換えた」行動を加点式で記録**し、練習の積み重ねを見えるようにします（ERP＝曝露反応妨害法／HRT・CBIT＝ハビット・リバーサルの考え方に基づく設計）。

> ⚠️ このアプリは**治療の代替ではなく補助**です。つらさが続くとき・悪化したときは、主治医や専門家に相談してください。

## 公開URL

https://eternitybios-dot.github.io/ERP/

## 主な機能

- **ワンタップ記録**：できたこと（反応しなかった・確認しなかった・置き換えた等）を1タップで加点記録。減点・ノルマ・ランキングなし
- **今日の練習**：自作の練習メニューから毎日1つ計画的曝露を提案（パス自由）
- **いま波が来てる**：呼吸ガイドで衝動の波をやり過ごす
- **数の契約をしない練習**：回数・数合わせの衝動から「降りる」ための画面（数・タイマー・完了を一切扱わない）
- **がんばりの記録**：累計ブレイブ・マイルストーン・練習の足あと（2値ドット）・発見ノート・月1セルフチェック
- **リマインダー**：1日1回のアプリ内通知＋確実に届くカレンダー（.ics）登録
- **オフライン対応**：一度開けば通信なしで動作

## データとプライバシー

- 記録はすべて**この端末のブラウザ内（localStorage）だけ**に保存されます。サーバーには何も送信されません
- そのため、**ブラウザ／ホーム画面アプリのデータを削除すると記録も消えます**
- 「履歴」タブの**バックアップ**でJSONファイルとして書き出し、**復元**で読み込めます（機種変更・再インストール前に推奨）
- バックアップには症状・メモ等の個人的な情報が平文で含まれます。**共有端末や共有フォルダに置かないでください**

## iPhoneでアプリとして使う

1. SafariでURLを開く
2. 共有ボタン →「ホーム画面に追加」
3. ホーム画面のアイコンから起動（全画面のアプリとして動作）

対応環境：iOS Safari（ホーム画面追加推奨）／その他モダンブラウザ（Chrome・Edge・Firefox最新版）

## 開発

ビルド不要の静的サイトです（HTML / CSS / Vanilla JS）。

```bash
# ローカル起動
python3 -m http.server 8080
# → http://localhost:8080

# テスト（加点ロジック・日付集計・復元の検証など）
node test/test.js
```

Service Worker の動作確認には `http://localhost` か `https://` が必要です（`file://` 不可）。

デプロイは `main` ブランチへのプッシュで GitHub Pages が自動更新します。

設計方針・データモデル・変更時の注意は [CLAUDE.md](CLAUDE.md) を参照してください。
